# Fleaxova

Fleaxova is a modern, full-stack freelance marketplace platform built with Next.js, React, and Firebase. It connects clients with skilled freelancers from around the world, enabling them to find talent and get work done efficiently.

## Key Features

*   **For Clients:** Post jobs, browse a marketplace of freelancer services, and hire the perfect talent for any project.
*   **For Freelancers:** Create a professional profile, list your services, and apply for jobs that match your skills.
*   **Core Functionality:** Secure user authentication, a comprehensive dashboard, service and job listings, a user wallet, and a messaging system for collaboration.
*   **Tech Stack:** Built with the Next.js App Router, TypeScript, Tailwind CSS, and ShadCN for the UI. It uses Firebase for backend services including Authentication and Firestore.
